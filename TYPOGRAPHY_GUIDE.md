# Guia de Tipografia - Nayara Estética

## **FONTE IMPLEMENTADA:**

### **1. Fontes Principais**
- **Primária:** Inter (sans-serif)
- **Display/Títulos:** Playfair Display (serif)
- **Corpo de Texto:** Lora (serif)

### **2. Hierarquia Tipográfica**

#### **Títulos Principais (Hero)**
```css
.text-hero
/* Aplicar no título principal "Nayara Andreassi" */
```

#### **Títulos de Seção**
```css
.text-section-title
/* Aplicar em "Nossos Serviços", "Sobre Nós", "Depoimentos" */
```

#### **Títulos de Cards**
```css
.text-card-title
/* Aplicar em títulos de serviços */
```

#### **Corpo de Texto Aprimorado**
```css
.text-body-enhanced
/* Aplicar em descrições de serviços e texto descritivo */
```

### **3. Classes de UI**

#### **Navegação**
```css
.nav-enhanced
/* Aplicar nos links de navegação */
```

#### **Botões**
```css
.btn-enhanced
/* Aplicar em todos os botões */
```

#### **Preços**
```css
.price-enhanced
/* Aplicar nos preços dos serviços */
```

### **4. Aplicação por Seção**

#### **Header/Hero Section**
- Título principal: `text-hero`
- Descrição: `text-body-enhanced`
- Links de navegação: `nav-enhanced`

#### **Seção de Serviços**
- Título da seção: `text-section-title`
- Títulos dos serviços: `text-card-title`
- Descrições: `text-body-enhanced`
- Preços: `price-enhanced`
- Botões: `btn-enhanced`

#### **Sobre**
- Título: `text-section-title`
- Texto do parágrafo: `text-body-enhanced`

#### **Depoimentos**
- Título: `text-section-title`
- Texto do depoimento: `testimonial-text`
- Autor: `testimonial-author`

#### **Contato**
- Título: `text-section-title`
- Horário: `text-body-enhanced`
- Botões: `btn-enhanced`

#### **Footer**
- Copyright: `footer-text`

## **BENEFÍCIOS DA NOVA TIPOGRAFIA:**

### **1. Sofisticação**
- Playfair Display adiciona elegância aos títulos
- Lora melhora a legibilidade do texto corrido

### **2. Profissionalismo**
- Inter é amplamente usada por empresas premium
- Pesos bem definidos para hierarquia clara

### **3. Performance**
- Fontes otimizadas do Google Fonts
- Carregamento eficiente com preconnect

### **4. Responsividade**
- Tamanhos responsivos com clamp()
- Adaptação automática a diferentes dispositivos

### **5. Acessibilidade**
- Alta legibilidade
- Contraste otimizado
- Espaçamento adequado

## **INSTRUÇÕES DE USO:**

2. **Aplicar classes conforme o guia acima**

3. **Testar em diferentes dispositivos**
